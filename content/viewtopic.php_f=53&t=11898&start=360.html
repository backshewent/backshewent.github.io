Old School Dead Tree Dictionary (American Heritage Dictionary of the English Language 3rd Edition--a wonderful reference everyone should own
If you're referring to the meat-like product you might eat in a sandwich, it's bologna, but baloney is a variant. If you're using a nice word for bullshit, you use baloney. The etymological relation between bologna and baloney in the second sense is uncertain, thus, using bologna in the sense you used it is non-standard.
