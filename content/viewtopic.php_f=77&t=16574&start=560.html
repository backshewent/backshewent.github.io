Got the Windows 10 upgrade.

Actually it's a lot better than Windows 8. The god-awful Metro Start screen or whatever they called it is gone (well, sort of, the god-awful Metro is still there but it's in a Start menu now, so it doesn't take up the whole screen anymore and there's a sort-of familiar list of applications again). The native apps no longer are full-screen only. Everything seems pretty snappy in general.

The biggest issue that I have is that in terms of user interface on a desktop PC I still cannot find any meaningful improvement over Windows 7.
