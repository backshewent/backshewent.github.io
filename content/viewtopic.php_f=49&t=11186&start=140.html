It might be a useless stat and it probably is but no one has really shown that it is. I'm pretty sure the Phillies always have a very good record when Rollins scores a run. I doubt if that's true for everyone.

What does it mean when Rollins scores a run in a game? It means he got on base and someone drove him in with a hit most of the time. So that means the top of the order got a few hits. So maybe when the top of the order gets a few hits they win most of the time.

I'd still like to see the records of all teams when their lead off guy scores a run for a number of years. That might tell us something.

As it is right now it is a useless stat because there is no context. Much like a few years ago when the writers made a big deal about Brett Myers really good ERA in the 11 games he won. Now of course, you can look up everyone's ERA in games they win and pretty much everyone has really good ERAs in games they win.

Without context, all stats are useless. With context you can tell a stats value. I'm pretty sure the team record when Rollins scores a run doesn't mean much. But I'd like to see it. (I don't even know where you look stuff like this up although I'm sure there is a place).
