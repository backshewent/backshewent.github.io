What makes a game miserable isn't length due to offense or extras (well, not usually), but when it grinds to a halt because each manager is changing pitchers 5, 6, 7 times. I think the way to handle this without interfering with the game itself is to simply limit the number of pitchers who can be active for a single game. So, before the game, each team must designate a maximum number of available pitchers... say 5 (starter plus 4 relievers). One of those relievers would need to be a long reliever, so you'd be down to 3 "specialists" which could not be used frivolously over every goddamn lefty/righty matchup.
