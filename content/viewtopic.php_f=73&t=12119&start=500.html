Pancakes, waffles and French toast all vary in breakfast deliciousness. The range for French toast is far greater though--the key to outstanding French toast is to use the best bread you can find, and it helps if it's ever so slightly stale. That baguette left over from last night's dinner is perfect. Also, you need to soak it in the egg mixture so that it's saturated but still holds together.
