I think that a lotta times when someone is talking about Irony, what they're talking about isn't really Irony. It might be Sarcasm or Satire or even just plain Sardonic.

Before the internets, we learned in school that O Henry (the writer not the candy bar) was always using irony. That's the guy we read when we were learning about Irony. So when I hear that something is ironic, I ax myself: "Does that seem like something O Henry might have written?"

If not, then it isn't irony.
