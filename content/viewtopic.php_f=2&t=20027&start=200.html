I can't make anything of the data. Using COVID tracker, here are the April trends.

- Average daily testing increase - 3.9%; average daily tests - 147,086
- Average daily positive increase - 1.1%; average daily positive tests - 29,030
- Average daily positive rate of new tests - 20.0%

The five days with the highest decreases in positive tests in April were either a weekend or a Monday. Outliers on the other end are Tuesdays and Wednesdays.

Since Thursday, we had five days in a row where new positive tests were less than the previous week, but that trend stopped yesterday.

Maybe, if the virus was here in January / February, there is a little herd immunity, but with testing remaining relatively flat, I am not sure if we can draw any conclusions beyond this feels like we're watching Sisyphus on loop.
