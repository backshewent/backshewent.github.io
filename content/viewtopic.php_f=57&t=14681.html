There's been some renewed buzz about the "Redskins" name for the Washington football club this week, which leads me to wonder what folks think about the above question. Essentially, if you could only change one, which would it be?

I find it strange because if the Redskins had wound up with Chief Wahoo, or the Cleveland Indians had been named the "Redskins," BOTH the "Redskins" and "Chief Wahoo" would have been gone long ago. But because the "Redskins" use the arguably most dignified Native American related logo, and the Indians use a word as their team nickname that has not been deemed in and of itself offensive (it was going that way for awhile when "Native American" was rising as the universally preferred term, but now "American Indian" has almost as much foothold), both survive.

I'm not sure myself, by the way. Ideally, both would go away, but I can see arguments for making either a priority, as they are both terrible.
