I set up a UniFi mesh with four hardwired access points and it works great. But I'd probably recommend the google system for most people. Looks much easier to manage and set up. The one downside to the google system is that you're married to their router (for all intensive purposes) so if there's a router feature that you really want or use and they don't have it you're SOL. For instance, port forwarding, advanced family controls, QOS, ReadyShare, whatever. Granted this is probably out of scope for your average home network, but should be considered in case it's not applicable to you.

by JFLNYC » Sun Apr 22, 2018 13:02:05

I was going to suggest MoCA for you. I’ve used it successfully many times where Ethernet outlets are not available but coax outlets are. From there it should be pretty easy to set up wireless access points using an Ethernet cable from the MoCA adapter into the access point. I’ve had good luck using Apple Airport Extremes/ Expresses in that situation. Just set them up with exactly the same SSID and passwords and you’ll have yourself a seamless web. I’m doing that here in our house (without the MoCA since the house has Ethernet outlets throughout). I have two AirPort Extremes plus an Airport Express on the lanai along with one other wireless extender given to me free from my security installer and we’re covered seamlessly both inside and out. Plus we have some of the more advanced options such as port forwarding and Back to My Mac. Of course you can use another brand besides Apple to achieve the same effect.
