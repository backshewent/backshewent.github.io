by Wolfgang622 » Sun Apr 19, 2020 09:16:02

I understand the running game has been sort of “debunked” by modern statistical analysis as well - that the value of a stolen base is small so you should only attempt it if the risk is correspondingly small - but its total disappearance is bad for baseball, and in any event expressing the value of an active running (and successful) running game purely in terms of the value of an extra base is to my mind one of the ways in which statistical analysis really does fail to take the whole picture into account. It’s not just the successful swipe of a bag, or the success rate, that counts. It’s also the disruption the threat brings, and while that can be harder to quantify, it is just as real. Against the Phillies, I and am sure many of you are still scarred for life by every time we played the goddamn 2003 Marlins. On everybody’s mind when Pierre and Castillo came up was “we have to get him out” because if we didn’t, a walk would be a guy on third with one out in the blink of an eye, and I am sure if WE thought that way, so too did the pitchers - and that this psychological weight explains a lot about the overall poor performance of the 2003 Phillies against those Marlins.

But in a more singular, concrete example that went for us instead of against us, look no further than the final game of the 2007 home sweep of the Mets. In that final inning, the fact that Wagner could not control the run game not only made scoring the winning run easier, it very clearly rattled him, successful or not (though it was successful in that case). Baseball, with its focus on individual performance, is, like golf, a game where the player can get dug inside his own head - because of things that are measurable, and things that are not. You don’t always have to steal a base to affect the outcome - but if there is no chance of stealing a base, there will be no effect.

All of this is to say: baseball might consider doing some rule change to encourage a return to teams taking the run game seriously, even if the value of an individual SB is low. It is exciting and it can drive pitchers to distraction, and distraction leads to mistakes, and mistakes lead to offense. Balanced against this, of course, is nobody likes incessant pick off throws, as they slow the game down.

Perhaps, then, for example, limiting the number of pick off throws per plate appearance to 3 would stimulate interest in the running game while not driving game time upwards.
