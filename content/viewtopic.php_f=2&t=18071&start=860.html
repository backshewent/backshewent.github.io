Re: Random Daze in a Summer Haze

by Bucky » Fri Sep 01, 2017 16:55:25

I chose mine based on the scientific method of W00T specials and Amazon deal-of-the-days. I've got four different models right now.

The last one I got is definitely the best- the one the video is from. I did pay like 49 bucks for that one on sale, which was a splurge for this project. And it was really frustrating for a long time- the pic was like 2-3 seconds behinds everything else and jumpy. But I lollygagged so I missed my return window, so I was stuck with it. And then a firmware upgrade made it much, much better.

https://smile.amazon.com/gp/product/B01 ... UTF8&psc=1


Other than that, everything else is older- TRENDnet TV-IP100W, D-Link DCS-930L, TRENDnet TV-IP651W.

Thew new one and the last one listed are PTZ- the others are static. If you want PTZ, I would recommend the Amcrest.

I use a free program on my pc called iSpy, which is pretty cool, and the 4 dollar android app which is called TinyCam Pro. It really helps to have router setup experience to map your internal IP addresses to external ports, and then use dynamic DNS to map your home IP address to a URL. That way you can view your cameras from anywhere- the same functionality comcast charges 40 bucks a month for.
